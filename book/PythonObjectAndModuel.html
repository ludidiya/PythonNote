Chapter 5 面向对象与模块

5.1 面向过程和面向对象

面向对象和面向过程都是开发程序的模式：

  • 面向过程：一系列函数按照某个过程顺次被调用，如：摘菜——洗菜——准备佐料——炒菜——出菜

  • 面向对象：找一个功能对象来完成整个过程，如：找个厨师能做上面所有事。面向对象的三大特征：封装、继承、多态。

5.2 类和对象

面向对象编程，是一种通过对象的方式，把现实世界映射到计算机模型的一种编程方法。

现实世界中，我们定义了“人”这种抽象概念，而具体的人则是“小明”、“小红”、“小军”等一个个具体的人。所以，“人”可以定义为一个类（class），而具体的人则是实例（instance）。

5.3 类的定义

5.3.1 基本语法

# 定义类
class Teacher(object):
  #定义类的功能/方法: 函数放在类里面以此来定义类的功能
  #方法1
  def eat(self):
    print('eat')
  #方法2
  def run(self):
    print('run')
    
# 根据类创建对象
t = Teacher()

# 用对象调用方法
t.eat()

# 给对象添加属性
t.name = 'XiaoMing'
t.age = 18

# 调用属性
print(t.name, t.age)

定义一个名为Teacher的类，继承于object（创建新的类要继承与其父类，而object是所有类的老祖宗）。

一个类里面可以定义多个功能/方法。
